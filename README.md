# Personal Website

Simple personal site built with React + Vite.

## Project Structure

- `src/components/HomePage.tsx`: Main page layout and presentation.
- `src/components/layout/SiteHeader.tsx`: Top navigation bar.
- `src/content/homePageContent.ts`: Editable content (about text, links, image paths).
- `public/images/`: Source images used by the site.
- `content/original-site-content-archive.md`: Archived copy of the original site text.
- `docs/`: Production build output (generated by `npm run build`).

## Editing Content

Update text and links in `src/content/homePageContent.ts`.

## Local Development

- `npm run dev` to run locally.
- `npm run build` to build production output into `docs/`.

## Notes

- `docs/` is generated output; avoid manual edits there.
- Keep source assets in `public/` and let build copy them into `docs/`.
